# LNU Assistant AI
 
LNU Assistant AI - це Telegram-бот для пошуку інформації в офіційній документації Львівського національного університету імені Івана Франка на основі підходу RAG.
 
## Про проєкт
 
Застосунок дозволяє студентам, викладачам та співробітникам університету швидко знаходити відповіді на запитання щодо навчального процесу безпосередньо у Telegram. Система автоматично збирає документи з офіційного сайту університету, обробляє їх та формує базу знань, на основі якої мовна модель генерує відповіді з посиланнями на конкретні документи та сторінки.
 
## Як це працює
 
1. Користувач надсилає запит у чат з ботом українською мовою.
2. Система знаходить найбільш релевантні фрагменти з університетських документів.
3. Мовна модель формує відповідь на основі знайденої інформації.
4. Разом із відповіддю користувач отримує посилання на джерела з номерами сторінок.
   
## Технології
 
- **Мова розробки:** Python
- **Інтерфейс:** Telegram (бібліотека aiogram)
- **Векторна база даних:** ChromaDB
- **Побудова RAG-ланцюга:** LangChain
- **Локальна мовна модель:** qwen3.5:2b через Ollama
- **Хмарна мовна модель:** glm-4.5-air:free через API OpenRouter 
- **Модель векторних представлень:** qwen3-embedding:0.6b
- **Оптичне розпізнавання тексту:** Tesseract (pytesseract)
- **Веб-скрейпінг:** requests, BeautifulSoup
- **Робота з PDF:** Poppler (pdf2image), pypdf, PyMuPDF, fpdf
